# 基本数据结构的实现

主要使用c++和python语言实现。

**在一个数据结构中，将会实现一下内容：**

* 创建：创建一个数据结构。
* 清除：删除数据结构。
* 插入：在数据结构指定的位置插入一个新元素。
* 删除：将数据结构中的某个元素删去。
* 搜索：在数据结构中搜索满足特定条件的元素。
* 更新：修改数据结构中某个元素的值。
* 访问：访问数据结构中的某个元素。
* 遍历：按照某种次序访问数据结构中的每一个元素，使每个元素恰好被访问一次。
